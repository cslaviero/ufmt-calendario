Start projeto

use os comandos abaixo:
usando o clone do projeto com a instalação do ambiente virtual
acesse o diretório do projeto:
exemplo: 
#
	cd c:/calendarioufr
#
ativar ambiente virtual
#
	venv\Scripts\activate
#
start projeto:
opcional Windows cmd:
#
	set FLASK_ENV=development 
	set FLASK_CONFIG=development
#
opcional Linux and Mac:
#
	export FLASK_ENV=development 
	export FLASK_CONFIG=development
#
obrigatório Windows cmd:
#
	set FLASK_APP=run.py 
	flask run
#
obrigatório Linux and Mac:
#
	export FLASK_APP=run.py 
	flask run
#
configurar start projeto-> link
	https://flask.palletsprojects.com/en/1.1.x/tutorial/factory/
----------------

start projeto sem a instalação do ambiente virtual
Installar o python
acesse o diretório do projeto:
exemplo: 
#
	cd c:/calendarioufr
#
instalação ambiente virtual:
#
	py -3 -m venv venv
#
start ambiente virtual:
#
	venv\Scripts\activate
#
----------------
